# Vibe B. -- Abgeschlossenes Uni-Projekt

Kuratierter Venue-Marketplace für Wien. Airbnb für Events -- Rooftops, Lofts, Studios, Gärten.

## Kernidee
Intent-based Search: User beschreibt sein Event in natürlicher Sprache, System gibt genau 3 kuratierte Matches zurück. Kein Filter-Grid, keine infinite Scroll. Die Einschränkung ist das Produkt.

## Was gebaut/entwickelt wurde
- Vollständiges Produkt- und Business-Modell (12% Host-Commission, zero Guest Fees)
- Design System: Warm Minimalism, Cormorant Garamond + DM Sans, Plum + Green
- Landing Page Struktur (10 Sektionen)
- Tech Stack definiert: React + Vite + Supabase + Stripe Connect + LLM Matching
- Investor Pitch: €35K Wandeldarlehen, Cap €800K, 20% Discount
- Pitch Script (EN) für WU Angel-Runde

## Was nie passiert ist
Kein einziges echtes Venue ongeboardet. Das war immer der eigentliche Bottleneck -- Supply vor allem anderen.

## Was mitgenommen werden kann
- Das "3 Matches statt infinite grid" Interface-Prinzip ist stark und wiederverwendbar
- Convertible Note Struktur für frühe Runden gut durchdacht
- Pitch-Logik: Moat ist Inventory, nicht AI -- übertragbar auf andere Projekte
